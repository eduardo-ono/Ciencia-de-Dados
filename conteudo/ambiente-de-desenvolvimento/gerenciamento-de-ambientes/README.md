## Python Pip

### Instalar Pacotes

* Supondo o arquivo `requirements.txt` salvo no diretório, executar no mesmo diretório:

```sh
pip install -r requirements.txt
```

* Exemplo do conteúdo de um arquivo `requirements.txt`:

```txt
pandas==0.25.3
numpy==1.17.4
```
